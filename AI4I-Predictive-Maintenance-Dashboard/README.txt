# AI4I Predictive Maintenance Dashboard

## Project Overview
This project demonstrates an event-based predictive maintenance analysis using the AI4I manufacturing dataset.  
The dashboard evaluates equipment reliability, downtime patterns, and operational efficiency through OEE computation and failure root cause analysis.

## Key Metrics
- Planned Minutes: 10,000
- Down Minutes: 339
- Availability: 96.61%
- MTTR: 1.09 minutes
- MTBF: 31.16 minutes
- OEE: 89.94%

## Analysis Performed
- Event-level downtime modeling
- OEE calculation (Availability × Performance × Quality)
- Downtime Pareto (80/20 rule)
- Failure root cause breakdown (HDF, OSF, PWF, TWF, RNF)
- Shift-based reliability comparison
- MTTR and MTBF computation

## Key Insights
- HDF is the leading root cause of downtime.
- Top 3 failure types account for approximately 80% of total downtime.
- Night shift shows elevated failure frequency.
- Average downtime per event is 1.09 minutes.

## Tools Used
- Power BI
- DAX
- Event-based data modeling
- Reliability analysis methods

## Dataset
AI4I 2020 Predictive Maintenance Dataset
